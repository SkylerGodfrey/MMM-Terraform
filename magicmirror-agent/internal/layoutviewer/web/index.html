How to edit

  1. Tap a module chip to pick it up. Tap any region to move it there.
  2. Type a CSS length (e.g. 420px, 30vh) in a region's height field. Leave blank to remove its cap.
  3. Tap a chip's EX badge to exempt it from its region's cap, or use the chip popover for tighter per-module sizing.
  4. Edits auto-save to the working copy. Tap Reset to live to discard, or Save to make them durable.

Legend

Capped — region has a max-height; overflow clips.
Suspended — an exempt module lifts the cap for the whole region.
Uncapped — region accepts whatever height its modules take.
Reserve — bottom 140 px held for the Scenes2 indicator strip.
